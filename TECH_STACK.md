# Tech Stack Specifications

## Core
- **Runtime:** Node.js (Latest LTS)
- **Package Manager:** pnpm (Preferred) or npm
- **Extension Framework:** Plasmo (`plasmo`)

## UI/UX
- **React:** v18+
- **Tailwind CSS:** v3.4+
- **Icons:** `lucide-react`
- **Components:** `@radix-ui/react-*` (select, dialog, tooltip)
- **Animations:** `framer-motion` (keep it light)

## Utilities
- **Markdown Rendering:** `react-markdown` (for LLM responses)
- **DOM Cleaning:** `@mozilla/readability`
- **Local AI:** `@mlc-ai/web-llm`

## Development
- **Linter:** ESLint + Prettier
- **Types:** TypeScript v5+
